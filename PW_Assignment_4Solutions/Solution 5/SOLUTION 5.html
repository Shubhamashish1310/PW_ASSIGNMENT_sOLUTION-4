Semantic Elements

HTML5 introduced several new semantic elements that provide a clearer structure to web pages. Some of the key elements include:

  • <header>: Represents a group of introductory or navigational aids.
  • <nav>: Contains navigation links.
  • <section>: Defines a section in a document, such as chapters, headers, footers, or any other grouping of content.
  • <article>: Represents a self-contained composition in a document.
  • <footer>: Defines a footer for a document or a section.
  • <aside>: Represents a portion of a document whose content is only indirectly related to the main content.
  • <figure> and <figcaption>: Used for embedding media, such as images, videos, or code snippets, along with captions.

Audio and Video🎵🎵🎧📽️📽️🎥

HTML5 provides native support for embedding audio and video content without the need for third-party plugins like Flash. This is done using the <audio> and <video> elements.

Canvas and SVG

HTML5 introduces <canvas> and <svg> elements for drawing graphics directly in the browser. The <canvas> element allows for bitmap-based rendering, while <svg> supports vector-based graphics.
